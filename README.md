## 🎯 What This Is

A **data-driven strategy portfolio** that goes beyond a standard resume. Built as a live, interactive web application to demonstrate:

- **SCQA Problem Framing** — McKinsey's Barbara Minto methodology applied to real cases
- **Interactive EBITDA Simulator** — Live 5-variable unit economics stress test (Swiggy Q-Commerce model)
- **Performance Analytics Dashboard** — Chart.js visualizations with animated KPIs
- **Forensic Case Studies** — Honasa supply chain audit, Swiggy dark-store economics, TruthGrid evaluation engine
- **AI Automation Portfolio** — Webhook CRM pipelines, n8n orchestration, Google Gemini API integration

## 🛠 Tech Stack

| Layer | Technology |
|-------|-----------|
| **Structure** | Semantic HTML5 |
| **Styling** | Vanilla CSS (custom design system, CSS variables, responsive grid) |
| **Interaction** | Vanilla JavaScript (IntersectionObserver, Chart.js) |
| **Charts** | Chart.js 4.4 (radar, line, doughnut, bar) |
| **Typography** | Playfair Display + Inter + JetBrains Mono |
| **Hosting** | GitHub Pages |

## ✨ Features

- **📊 Live EBITDA Simulator** — Adjust 4 operational variables to model per-order contribution margin
- **📈 Animated KPIs** — Counter animations triggered by scroll intersection
- **🎨 Dark/Light Sections** — Consulting-grade visual hierarchy with saffron accent palette
- **📱 Fully Responsive** — Mobile nav drawer with overlay, adaptive grids, print-optimized
- **♿ Accessible** — Skip-to-content link, semantic HTML, keyboard-navigable tabs
- **🔍 SEO Optimized** — Open Graph, Twitter Cards, structured meta tags
- **🎭 Scroll Reveal Animations** — Staggered element reveals with cubic-bezier easing
